mcuxClMac_Mode_t mcuxClMac_Mode_HMAC_SHA2_256_CSS
static

HMAC mode.

The input buffer in will be modified by applying padding to it. The caller must ensure that the input buffer is large enough to hold this padding. The total buffer size including padding can be calculated using the macro MCUXCLMAC_GET_HMAC_INPUTBUFFER_LENGTH on the data size inLength.

Also note that mcuxClMac_Mode_HMAC_SHA2_256_CSS only works with keys loaded into coprocessor (see mcuxClKey for details).
