const mcuxClRandom_ModeDescriptor_t mcuxClRandom_mdCSS_Drbg

Mode for a DRBG implemented by the CSS.

This mode provides a function to get random values from a DRBG implemented by the CSS. The provided security strength is HW dependent and can be determined using the function mcuxClRandom_checkSecurityStrength
