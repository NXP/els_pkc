Deprecated type for mcuxClSession protected status codes.

Deprecated type for error codes used by code-flow protected Mac component functions.

Deprecated type for Key component error codes, returned by functions with code-flow protection.

Deprecated type for mcuxClEcc component return codes.

Deprecated type for CSS driver protected status codes.

Set the requested key properties of the destination key.

Key-pair generation function.

This function can be used to perform a key-pair generation operation.

Note: the key handles privKey and pubKey must already be initialized and contain a proper key type (matching to the generation algorithm), protection mechanism and key data buffers.

Parameters
pSessionHandle for the current CL session.
generationKey generation algorithm that determines the key data stored in privKey and pubKey.
[out]privKeyKey handle for the generated private key.
[out]pubKeyKey handle for the generated public key.
[out]pPrivDataLengthWill be incremented by the number of bytes of data that have been written to the key data buffer of privKey.
[out]pPubDataLengthWill be incremented by the number of bytes of data that have been written to the key data buffer of pubKey.
Returns
status

Key descriptor initialization function including applying a protection mechanism.

This function performs the initialization of a Key descriptor. In addition the given protection mechanism gets applied to the given raw key data.

Parameters
pSessionHandle for the current CL session.
protectionProtection mechanism to be applied to the given pPlainKeyData.
protectedKeyKey to be initialized and protected.
typeType of the key.
[in]pPlainKeyDataPlain raw key data.
plainKeyDataLengthNumber of bytes available in the pPlainKeyData buffer.
[in]pAuxDataAuxilary data needed for the given key protection.
auxDataLengthNumber of bytes available in the pAuxData buffer.
[out]pProtectedKeyDataProtected raw key data (after applying protection to the pPlainKeyData)
