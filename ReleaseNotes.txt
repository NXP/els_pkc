--- New Features 
   - EDDSA functionality included
   - Bugfixes
   
--- Features:
   - Components included: mcuxClEls
   - Components included: mcuxClAead, mcuxClCipher, mcuxClHash, mcuxClKey, mcuxClMac, mcuxClMemory, mcuxClPadding, mcuxClSession
   - Components included: mcuxClPkc, mcuxClMath, mcuxClEcc, mcuxClPrng, mcuxClRandom, mcuxClRsa, mcuxClTrng
   - Components included: mcuxClPsaDriver
   - Components included: mcuxCsslCPreProcessor, mcuxCsslFlowProtection, mcuxCsslMemory, mcuxCsslParamIntegrity, mcuxCsslSecureCounter
   - mcuxClEls supports the following functionality
        - AES-128/192/256 encryption/decryption, modes ECB/CBC/CTR/GCM/CMAC
        - SHA2-224/256/384/512 (including sha-direct mode)
        - KDF: CKDF (NIST SP 800-108 / SP800-56C), HKDF (RFC5869)
        - HMAC
        - ECC Sign/Verify/Keygen/Keyexchange (P-256)
        - RFC3394 key wrapping/unwrapping
        - Key Import/Export/Utilities/Provisioning
        - Random number Generation (DRBG/PRNG)
        - TLS Master/Session Key generation

--- Known Issues Limitations:
   - psa/crypto_struct.h needs to be modified
   - toolchain.h renaming still to be done
   - ip_platform.h including fsl_device_registers.h to be done
   - occasional timeouts/iterations exceeded for RSA key generation (this is due to statistical variation)
   - failing test case on internal function for extracting TRNG values
   - dma final address readback checks are disabled on this release
